Detailed Description

A GroupMultiplierBC class handles boundary conditions.

The GroupMultiplierBC class is in charge of applying boundary conditions to the system. The GroupMultiplierBC class only takes care of homogeneous Dirichlet conditions. Non-homogeneous displacement boundary conditions are treated as Load so that can be solved iteratively. Others are handled by general constraint class such as MPC. The GroupMultiplierBC class stores the boundary condition category, type, node(s) and corresponding DoF(s). The Domain invokes process(const shared_ptr<Domain>&) method to modify the global stiffness matrix.
