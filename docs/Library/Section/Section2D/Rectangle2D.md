# Rectangle2D

2D Rectangle Section

## Syntax

```
section Rectangle2D (1) (2) (3) (4) [5] [6]
# (1) int, unique tag
# (2) double, section width
# (3) double, section height
# (4) int, material tag
# [5] int, number of integration points, default: 6
# [6] double, eccentricity measured from the centre, default: 0.0
```
