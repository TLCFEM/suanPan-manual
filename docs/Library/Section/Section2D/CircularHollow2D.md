# CircularHollow2D

2D Hollow Circular Section

## Syntax

```
section CircularHollow2D (1) (2) (3) (4) [5] [6]
# (1) int, unique section tag
# (2) double, radius
# (3) double, thickness
# (4) int, material tag
# [5] int, number of integration points, default: 10
# [6] double, eccentricity, default: 0.0
```
