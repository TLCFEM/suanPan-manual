# MinResistance

The `MinResistance` criterion tests the resistance of the specified node, if it is **smaller** than the given value, the
analysis is terminated.

## Syntax

```
criterion MinResistance (1) (2) (3) (4)
# (1) int, unique criterion tag
# (2) int, node
# (3) int, dof
# (4) double, limit
```
