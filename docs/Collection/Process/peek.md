# peek

The `peek` command can be used to print out predefined information. This command has no effect if global screen print is
switched off.

## Syntax

```
peek node (1)
peek element (1)
peek material (1)
peek constraint (1)
peek recorder (1)
peek solver (1)
peek integrator (1)
peek group (1)
peek amplitude (1)
# (1) int, object tag

peek eigenvalue
```
